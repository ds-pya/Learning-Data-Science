1. 개요

본 서비스는 사용자가 스크린샷·사진 캡처 등을 통해 수집한 다양한 정보(메신저 대화, 예약 페이지, 초대장, 콘서트 포스터 등)로부터 일정을 자동으로 추출해 캘린더에 저장해주는 “캘린더 박제 자동화” 기능이다. 이미지 속 텍스트를 OCR로 인식한 뒤, 이미지와 텍스트를 함께 분석해 이벤트명·날짜·시간·장소 등을 추출하고, 이를 캘린더에 자동 생성하거나 제안하는 방식으로 동작한다. 이를 통해 사용자는 번거로운 일정 입력 없이, 캡처만으로도 정확한 일정 관리가 가능해진다.

2. 필요성

현대 사용자들은 메신저, SNS, 웹 예약 시스템 등 다양한 채널에서 약속과 일정 정보를 얻지만, 이를 캘린더에 직접 옮기는 과정은 반복적이고 귀찮은 작업이다. 특히 초대장·예약페이지·포스터 등 이미지 형태의 정보는 텍스트 복사도 어렵고, 날짜·장소를 일일이 입력해야 하므로 누락·지연·오입력의 가능성이 높다. 이러한 흐름 속에서 이미지 기반 일정 자동 추출 기능은 사용자의 일정 관리 부담을 줄이고 정확도를 높여, 실질적인 편의성과 사용 경험을 개선할 필요성이 높다.

3. 개발 기획

서비스는 크게 세 단계로 구성된다: (1) 이미지에서 공통 OCR 수행, (2) OCR 텍스트와 이미지 임베딩을 함께 입력으로 사용하는 멀티모달 모델을 통해 문서 타입 분류(초대장/예약/포스터/채팅 등)와 일정 관련 정보 추출(이벤트명·날짜·시간·장소·인물)을 동시에 처리, (3) 추출된 정보를 기반으로 캘린더 이벤트 카드 생성 후 사용자에게 자동 제안하는 UX 제공. 초기 단계에서는 예약·초대장·포스터 중심으로 지원하고, 이후 채팅 기반 약속 추출, 기간형 이벤트 처리, 반복 일정 추론 등으로 확장한다. 모든 분석은 온디바이스 환경에서 수행하여 개인정보 보호와 즉시 처리 성능을 만족한다.

4. 시나리오 및 기대효과

사용자는 카카오톡 캡처, 모바일 청첩장 이미지, 미용실 예약 페이지 캡처 등 어떤 이미지든 앱으로 공유하거나 자동 감지 기능을 통해 전달한다. 시스템은 이미지에서 텍스트를 읽고 일정 정보를 추출하여 “◯◯ 결혼식 — 11월 30일 14:00, ◯◯웨딩홀”과 같은 이벤트 카드로 제안하며, 사용자는 한 번의 탭으로 캘린더에 저장한다. 이를 통해 일정 입력의 번거로움을 제거하고, 약속 누락·지연·오타를 줄일 수 있으며, 사용자는 다양한 채널에 흩어진 일정 정보를 자동으로 간편하게 통합·관리하는 경험을 얻게 된다.
